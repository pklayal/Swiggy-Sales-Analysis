# 🍔 Swiggy Sales Data Analysis

## 📌 Project Overview
This project analyzes Swiggy sales data to extract insights on:
- Sales trends
- Customer behavior
- Food preferences
- Geographic performance

## 📊 Key KPIs
- Total Sales
- Average Rating
- Average Order Value
- Rating Count
- Total Orders

## 📈 Analysis Performed
- Monthly & Daily Sales Trends
- Veg vs Non-Veg Analysis
- State & City-wise Sales
- Customer Rating Analysis
- Weekly & Quarterly Trends

## 🛠 Tools Used
- Microsoft Excel
- Data Visualization
- Pivot Tables

## 📁 Files
- Dataset (Excel)
- Dashboard
- Project Report

## 🚀 Insights
- Weekend sales are higher
- Veg items dominate orders
- Certain states contribute major revenue
