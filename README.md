## 📌 Project Overview
This project presents a **comprehensive analysis of Swiggy sales data** to uncover actionable insights related to:

- 📈 Sales performance trends  
- 👥 Customer behavior & engagement  
- 🍽 Food preferences (Veg vs Non-Veg)  
- 🌍 Geographic performance (State & City-wise)  

The project uses **Excel-based data analysis and dashboard visualization** to simulate real-world business intelligence workflows.

---

## 🎯 Objectives
- Analyze sales patterns across time (daily, monthly, quarterly)
- Identify high-performing regions and food categories
- Evaluate customer satisfaction using ratings
- Build an interactive dashboard for decision-making

---

## 📊 Key Performance Indicators (KPIs)

| KPI | Description |
|-----|------------|
| 💰 Total Sales | Overall revenue generated |
| ⭐ Average Rating | Customer satisfaction level |
| 🧾 Average Order Value | Revenue per order |
| 📊 Rating Count | Customer engagement |
| 📦 Total Orders | Total number of orders |
